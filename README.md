<h1 align="center">Kho ảnh</h1>

<p align="center">
Kho lưu trữ ảnh, GIF và video cho Scratch, HTML và các dự án khác.
</p>

<hr>

<h2>Thư mục</h2>

<pre>
images/
</pre>

<p>
Tất cả file nên được upload vào thư mục <b>images</b>.
</p>

<hr>

<h2>Hỗ trợ</h2>

<h3>Ảnh</h3>

<ul>
<li>PNG</li>
<li>JPG</li>
<li>JPEG</li>
<li>WEBP</li>
<li>GIF</li>
<li>SVG</li>
</ul>

<h3>Video</h3>

<ul>
<li>MP4</li>
<li>WEBM</li>
<li>MOV</li>
<li>OGG</li>
</ul>

<hr>

<h2>Link file</h2>

<p>Mẫu đường dẫn:</p>

https://TEN_GITHUB.github.io/TEN_REPO/images/TEN_FILE
</pre>

<p>Ví dụ:</p>

<pre>
https://TEN_GITHUB.github.io/TEN_REPO/images/logo.png
</pre>

<hr>

<h2>Thêm file mới</h2>

<ol>
<li>Mở thư mục <b>images</b></li>
<li>Upload file</li>
<li>Commit changes</li>
</ol>

<p>
File mới sẽ tự động xuất hiện trên trang web sau vài giây.
</p>
